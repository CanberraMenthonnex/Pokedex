# POKEMON POKEDEX WEB APPLICATION 

## Description
This application allow the user to search for some data about a Pokemon thanks to the [Poke Api](https://pokeapi.co/)


## How to install
<hr><br>

```
npm install
```
```
npm start
```

## How to build
<hr><br>

```
npm run build
```

## How to open
<hr><br>

Open http://localhost:3000

## Requirements
<hr><br>

- node version >= 14.0.0
- npm version > 5.6
